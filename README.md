# raymondriter.github.io

Static site root. Hosts the Tesla virtual-key public key at:

  https://raymondriter.github.io/.well-known/appspecific/com.tesla.3p.public-key.pem

Used by [developer.tesla.com](https://developer.tesla.com) to verify the
public half of the keypair whose private half signs commands sent to my
Tesla via the Fleet API.

See the [hey-nabu-climate-concierge](https://github.com/Raymondriter/hey-nabu-climate-concierge) repo for
how this key is generated and used.
